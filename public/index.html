Responsive Blog Project

A responsive blog page with max-width media queries. I focused on flexibility so that it is adaptable to any current or future device.

Images are set to a max-width of 100% and auto height. As the page shrinks, the sidebar eases in until it changes to 100% at 710 pixels.

The responsive nav is hidden until 767 pixels, then displayed as inline. jQuery uses the class, .show, to transition ease the nav ul height.

HTML5, CSS3
